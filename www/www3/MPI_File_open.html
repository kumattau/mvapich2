MPI_File_open

 Opens a file

Synopsis

int MPI_File_open(MPI_Comm comm, char *filename, int amode, 
                  MPI_Info info, MPI_File *fh)

Input Parameters

comm
communicator (handle)
filename
name of file to open (string)
amode
file access mode (integer)
info
info object (handle)

Output Parameters

fh
file handle (handle)
